Bewegingsblokken zorgen ervoor dat Eliobot in de ruimte kan bewegen.

Bewegingsblokken per vierkant

Ga X velden vooruit​

Type: Bestelling

Definitie :

Met dit blok kan Eliobot een bepaald aantal vakjes vooruitgaan.

Gebruik :

Als je Eliobot een bepaalde afstand vooruit wilt verplaatsen.

Draai rechts of links​

Type: Bestelling

Definitie :

Met dit blok kan Eliobot naar rechts of links draaien (90 graden).

Gebruik :

Als je Eliobot wilt gebruiken.

Vrije bewegingsblokken

Ga vooruit​

Type: Bestelling

Definitie :

Met dit blok kun je Eliobot vooruit of achteruit bewegen.

Gebruik :

Als u Eliobot vooruit of achteruit wilt bewegen.

Draai naar rechts​

Type: Bestelling

Definitie :

Met dit blok kan Eliobot naar rechts of links draaien (90 graden).

Gebruik :

Als je Eliobot wilt gebruiken.

Roteer met een vooraf gedefinieerde hoek​

Type: Bestelling

Definitie :

Met dit blok kan Eliobot onder een precieze hoek roteren.

Gebruik :

Als je Eliobot iets naar rechts of links wilt laten draaien.

Stop de motoren​

Type: Bestelling

Definitie :

Met dit blok kun je de motoren van Eliobot stoppen.

Gebruik :

Als je Eliobot wilt stoppen.

Draai een enkel wiel​

Type: Bestelling

Definitie :

Met dit blok kun je één Eliobot-wiel laten draaien.

Gebruik :

Als je slechts één Eliobot-wiel wilt laten draaien. Het is een alternatief voor rotatie.

Verander snelheid​

Type: Bestelling

Definitie :

Met dit blok kun je de snelheid van Eliobot (in percentage) wijzigen.

notitie

100% is de maximale snelheid.

Gebruik :

Als je wilt dat Eliobot sneller of langzamer gaat. Vertraag bijvoorbeeld als een sensor een obstakel detecteert.
