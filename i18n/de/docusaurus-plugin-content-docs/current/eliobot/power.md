## Verwendung mit Elioblocs

### Schalten Sie Eliobot ein

Um Eliobot einzuschalten, positionieren Sie den Knopf einfach in der Position `ON`.

### Eliobot ausschalten

Zum Ausschalten nach dem gleichen Prinzip positionieren Sie einfach den Knopf in der Position `OFF`.

## Verwendung mit Python

Wenn Eliobot an einen Computer angeschlossen ist, ist er halb eingeschaltet, Sie können ihn Befehle ausführen lassen, Programme darauf hochladen, er führt sie aus, aber alle an die Batterie angeschlossenen Elemente (Motoren) werden nicht mit Strom versorgt.

Um alle Funktionen von Eliobot nutzen zu können, müssen Sie den Netzschalter in den `ON` Modus versetzen.
