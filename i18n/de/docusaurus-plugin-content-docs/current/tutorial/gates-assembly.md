# Construire des obstacles

Das war's, Ihr Roboter ist zusammengebaut? Was wäre, wenn ich Ihnen sagen würde, dass sich am Ende Ihrer Schachtel möglicherweise noch eine Überraschung befindet!

Heben Sie die oberen Kartonblätter an, um ein letztes Blatt mit abgerundeten Ecken an der Unterseite des Kartons zu finden.

![alt text](/img/tutorial/gates-assembly/71490697-3D6C-4291-BBF0-2768DA5ADE4C_1_102_a.jpeg)

Nehmen Sie diesen Teller und teilen Sie ihn in zwei Teile.

![alt text](/img/tutorial/gates-assembly/9046FC62-53AD-45C0-8F4D-DE7E04C9D8B7_1_201_a-2048x1536.jpeg)

Dann lösen Sie die beiden Elemente in der Mitte jeder Halbplatte, das sind die Füße.
Das andere „U“-förmige Element ist die Tür

![alt text](/img/tutorial/gates-assembly/B7E39673-4AF7-4B1D-9C91-0DD8B087FE19_1_102_a.jpeg)

Montieren Sie die Beine an der Tür

![alt text](/img/tutorial/gates-assembly/C93D9ED2-DB41-4868-9806-7753AC228BA6_1_102_a.jpeg)

Und voilà! Sie haben gerade zwei kleine Türen erstellt, durch die sich Eliobot bewegen kann.

![alt text](/img/tutorial/gates-assembly/47A67EA2-FC0A-4CC5-B111-84B9F697D888_1_102_a.jpeg)

Sie können dies auch nutzen, um Ihre Hindernisse zu personalisieren

![alt text](/img/tutorial/gates-assembly/CC591BFB-BC38-47A7-9708-5E2B2DA3F492_1_201_a-2048x1536.jpeg)

Erfahren Sie in den folgenden Tutorials, wie Sie Eliobot so programmieren, dass er durch diese Türen geht.
