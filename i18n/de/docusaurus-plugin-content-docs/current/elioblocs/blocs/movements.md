Bewegungsblöcke ermöglichen es, Eliobot im Raum zu bewegen.

# Bewegungsblöcke pro Quadrat

## X Leerzeichen vorwärts bewegen

![X Fälle verschieben](/img/elioblocs/blocs/movements/avancer-cases.png)

**Typ**: Bestellung

**Definition**:

Dieser Block ermöglicht es Eliobot, eine bestimmte Anzahl von Feldern vorzurücken.

**Verwenden** :

Wenn Sie Eliobot eine bestimmte Distanz vorwärts bewegen möchten.

## Biegen Sie rechts oder links ab

![Nach rechts oder links abbiegen](/img/elioblocs/blocs/movements/tourner-cases.png)

**Typ**: Bestellung

**Definition**:

Dieser Block ermöglicht es Eliobot, sich nach rechts oder links (90 Grad) zu drehen.

**Verwenden** :

Wenn Sie Eliobot ausführen möchten.

# Freie Bewegungsblöcke

## Gehe vorwärts

![Vorwärts oder rückwärts bewegen](/img/elioblocs/blocs/movements/deplacer.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie Eliobot vorwärts oder rückwärts bewegen.

**Verwenden** :

Wenn Sie Eliobot vorwärts oder rückwärts bewegen möchten.

## Nach rechts schwenken

![Frei drehen](/img/elioblocs/blocs/movements/pivoter.png)

**Typ**: Bestellung

**Definition**:

Dieser Block ermöglicht es Eliobot, sich nach rechts oder links (90 Grad) zu drehen.

**Verwenden** :

Wenn Sie Eliobot ausführen möchten.


## Um einen vordefinierten Winkel drehen

![In einen Winkel drehen](/img/elioblocs/blocs/movements/pivoter-degres.png)

**Typ**: Bestellung

**Definition**:

Dieser Block ermöglicht es Eliobot, sich in einem präzisen Winkel zu drehen.

**Verwenden** :

Wenn Sie Eliobot dazu bringen möchten, sich leicht nach rechts oder links zu drehen.

## Stoppen Sie die Motoren

![Motoren stoppen](/img/elioblocs/blocs/movements/arreter-moteurs.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie die Motoren von Eliobot stoppen.

**Verwenden** :

Wenn Sie Eliobot stoppen wollen.

## Drehen Sie ein einzelnes Rad

![Nur ein Rad drehen](/img/elioblocs/blocs/movements/tourner-roue.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie ein einzelnes Eliobot-Rad drehen.

**Verwenden** :

Wenn Sie nur ein Eliobot-Rad drehen möchten. Es ist eine Alternative zur Rotation.

## Geschwindigkeit ändern

![Geschwindigkeit ändern](/img/elioblocs/blocs/movements/modifier-vitesse.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie die Geschwindigkeit von Eliobot (in Prozent) ändern.

:::note
100 % ist die maximale Geschwindigkeit.
:::

**Verwenden** :

Wenn Sie möchten, dass Eliobot schneller oder langsamer fährt. Verlangsamen Sie beispielsweise die Geschwindigkeit, wenn ein Sensor ein Hindernis erkennt.
