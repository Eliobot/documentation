In Elioblocs gibt es drei große Blockkategorien:
- Boolesche Blöcke
- Werteblöcke
- Befehlsblöcke

## Boolesche Blöcke

Mit booleschen Blöcken können Sie Tests und Vergleiche durchführen. Sie geben `wahr` oder `falsch` zurück.
Sie werden durch rautenförmige Blöcke dargestellt.

![Boolesche Blöcke](/img/elioblocs/blocs/blocs-type/boolean.jpg)

## Werteblöcke

Mit Werteblöcken können Sie Zahlen, Texte und Listen manipulieren.
Sie werden durch Blöcke in Form eines horizontalen Ovals dargestellt.

![Werteblöcke](/img/elioblocs/blocs/blocs-type/value.jpg)

## Befehlsblöcke

Mit Befehlsblöcken können Sie Aktionen ausführen. Sie werden auf unterschiedliche Weise durch rechteckige Blöcke dargestellt.

![Befehlsblöcke](/img/elioblocs/blocs/blocs-type/action.jpg)

![Befehlsblöcke 2](/img/elioblocs/blocs/blocs-type/action-2.jpg)

![Befehlsblöcke 3](/img/elioblocs/blocs/blocs-type/action-3.jpg)
