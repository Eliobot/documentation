Mathematikblöcke ermöglichen grundlegende mathematische Operationen. Sie sind sehr nützlich, um Berechnungen durchzuführen.

## Ganzzahl

![123](/img/elioblocs/blocs/maths/123.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie eine Ganzzahl definieren.

**Verwenden** :

Wenn wir eine Ganzzahl definieren möchten (z. B. in einem anderen Block).

## Grundlegende Operationen

![1 + 1](/img/elioblocs/blocs/maths/addition.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie Folgendes erreichen:
- Ergänzung
- Subtraktion
- Multiplikation
- Abteilung
- Macht

**Verwenden** :

Wenn wir eine einfache mathematische Operation ausführen möchten.

## Grundfunktionen

![Quadratwurzel aus 9](/img/elioblocs/blocs/maths/racine-carree.png)

- absoluter Wert
- Verneinung

- Logarithmus zur Basis 10

- Zehnerpotenz

## Trigonometrie

![sin 45](/img/elioblocs/blocs/maths/trigo.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie trigonometrische Funktionen berechnen:
- Nebenhöhlen
- Kosinus
- Tangente
- Arkussinus
- Arkuskosinus
- Arcus-Tangens

**Verwenden** :

Wenn wir trigonometrische Berechnungen durchführen möchten, um Eliobot beispielsweise Dreiecke zeichnen zu lassen.

## Konstanten

![pi](/img/elioblocs/blocs/maths/pi.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie den Wert mathematischer Konstanten abrufen:
- pi
-e
- Phi
- Wurzel von 2
- Wurzel von 1/2
- unendlich

**Verwenden** :

Erweiterte mathematische Berechnungen (Umfang eines Kreises, Fläche eines Dreiecks usw.).

## Testen Sie eine Ganzzahl

![Testen Sie, ob eine Zahl gerade ist](/img/elioblocs/blocs/maths/test-pair.png)

**Typ**: Boolescher Block

**Definition**:

Mit diesem Block können Sie testen, ob eine Zahl:
- sogar
- seltsam
- positiv
- negativ
- ganz
- zuerst
- teilbar durch

**Verwenden** :

Wenn wir eine Zahl testen möchten, um eine Aktion basierend auf ihrem Wert auszuführen.

## Runde

![Eine Zahl runden](/img/elioblocs/blocs/maths/arrondir.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie eine Zahl runden:
- auf die nächste ganze Zahl
- zur höheren ganzen Zahl
- zur unteren Ganzzahl

**Verwenden** :

Wenn wir eine Zahl runden möchten, um sie beispielsweise als Anzahl der Iterationen zu verwenden, können wir 2 oder 3 Iterationen durchführen, aber nicht 3,33.

## Listenvorgang

![Listensumme](/img/elioblocs/blocs/maths/somme-liste.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie Operationen an Listen ausführen:
- Summe
- Durchschnitt
- Minimum
- maximal
- Mittelwert
- Mehrheit
- Standardabweichung
- Zufallselement

**Verwenden** :

Wenn wir Berechnungen für eine Liste von Zahlen durchführen möchten. Oder wenn wir ein zufälliges Element aus einer Liste abrufen möchten.

## Rest einer Division

![Division mit Rest](/img/elioblocs/blocs/maths/reste-division.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie den Rest einer Division (Modulo) wiederherstellen.

**Verwenden** :

Wenn wir den Rest einer Division zurückgewinnen wollen, um eine Aktion basierend auf diesem Rest durchzuführen.

## Beschränke eine Ganzzahl zwischen zwei Werten

![Eine Zahl einschränken](/img/elioblocs/blocs/maths/contraindre-chiffre.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie eine Zahl zwischen zwei Werten einschränken.
Dadurch wird sichergestellt, dass die Zahl innerhalb eines vorgegebenen Bereichs liegt.

**Verwenden** :

Wenn wir sicherstellen möchten, dass eine Zahl in einem bestimmten Intervall liegt, bevor wir sie verwenden. Wenn wir beispielsweise Prozentsätze verwenden, möchten wir sicherstellen, dass die Zahl zwischen 0 und 100 liegt.

##atan2

![atan2](/img/elioblocs/blocs/maths/atan2.png)

**Typ**: Wert

**Definition**:

Mit diesem Block können Sie den Arkustangens zweier Zahlen berechnen.
Es wird verwendet, um den Winkel zwischen zwei Punkten zu ermitteln.

**Verwenden** :

Wenn wir den Winkel zwischen zwei Punkten berechnen möchten, drehen wir beispielsweise Eliobot.
