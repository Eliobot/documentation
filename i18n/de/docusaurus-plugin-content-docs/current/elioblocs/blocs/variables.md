Variablen ermöglichen es, Werte zu speichern und später im Programm wiederzuverwenden. Sie sind sehr nützlich, um Daten aller Art zu speichern.

## Erstellen Sie eine Variable

![Variablenerstellung](/img/elioblocs/blocs/variables/creer-variable.png)

Um eine Variable zu erstellen, klicken Sie einfach auf die Schaltfläche und es öffnet sich ein Fenster zur Eingabe des Variablennamens.

![Variablenname festlegen](/img/elioblocs/blocs/variables/nom-variable.png)

Geben Sie ihm einen Namen Ihrer Wahl, er muss eindeutig sein.

## Rufen Sie eine Variable auf

![Verwenden einer Variablen](/img/elioblocs/blocs/variables/appel-variable.png)

**Typ**: Wert

**Definition**:

Eine Variable kann überall im Programm aufgerufen werden, sofern sie zuvor deklariert wurde.

**Verwenden** :

Wenn wir eine Variable verwenden möchten, um einen Wert zu speichern und ihn später wiederzuverwenden.

## Eine Variable ändern

![Variableneinstellungen](/img/elioblocs/blocs/variables/change-variable.png)

**Definition**:

Wir können:
- Ändern Sie den Variablennamen
- Variable löschen

**Verwenden** :

Wenn wir eine bereits erstellte Variable ändern möchten.

### Auf einen bestimmten Wert setzen

![Variable auf bestimmten Wert setzen](/img/elioblocs/blocs/variables/definir-variable.png)

**Typ**: Bestellung

**Definition**:

Setzen Sie eine Variable auf einen bestimmten Wert. Dieser Wert kann eine Zahl, ein Text, eine Liste usw. sein.

**Verwenden** :

Wenn Sie eine Variable für einen Wert definieren möchten, um sich beispielsweise die Position eines Elements in einer Liste zu merken.

### Inkrement

![Variableninkrement](/img/elioblocs/blocs/variables/incrementer-variable.png)

**Typ**: Bestellung

**Definition**:

Beim Erhöhen einer Variablen wird der Wert einer Variablen um den gewünschten Wert erhöht.

**Verwenden** :

Wenn wir beispielsweise eine Variable bei jeder Schleifenumdrehung um 1 erhöhen möchten.
