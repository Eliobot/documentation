Eliobot ist mit einem Summer ausgestattet, mit dem sich Musiknoten abspielen lassen.

## Spielen Sie eine Note

![Note abspielen](/img/elioblocs/blocs/sounds/jouer-note.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie eine Musiknote spielen. Die Note wird durch ihren Namen (Do, Ré, Mi, Fa, Sol, La, Si), ihre Oktave und die Zeit, in der sie gespielt wird, definiert.

**Verwenden** :

Wenn Sie Ihre eigene Musik machen möchten.

## Spielen Sie eine Frequenz ab

![Wiedergabehäufigkeit](/img/elioblocs/blocs/sounds/jouer-frequence.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie eine Frequenz abspielen. Die Frequenz wird in Hertz (Hz) und die Abspieldauer angegeben.

**Verwenden** :

Wenn Sie eine bestimmte Frequenz abspielen möchten.

## Schweigen

![Spiel Stille](/img/elioblocs/blocs/sounds/silence.png)

**Typ**: Bestellung

**Definition**:

Dieser Block ermöglicht es Ihnen, eine Pause von der Musik einzulegen. Die Pausenzeit wird in Millisekunden definiert.

**Verwenden** :

Wenn Sie eine Pause von der Musik machen möchten, die Sie komponieren.

## Lautstärke ändern

![Lautstärke ändern](/img/elioblocs/blocs/sounds/modifier-volume.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie die Lautstärke des Summers ändern. Das Volumen wird als Prozentsatz definiert.

**Verwenden** :

Verringern Sie die Lautstärke, um andere nicht zu stören.

## Spielen Sie voreingestellte Musik ab

![Vordefinierte Musik abspielen](/img/elioblocs/blocs/sounds/jouer-musique.png)

**Typ**: Bestellung

**Definition**:

Mit diesem Block können Sie vordefinierte Musik abspielen. Es stehen mehrere vordefinierte Musik zur Verfügung.
- **Imperial March**: Die Musik von Darth Vader in Star Wars.
- **Totally Spies**: Der Ruf von Sam, Clover und Alex.
- **Polizei**: Die Polizeisirene.
- **Fluch der Karibik**: Die Musik von Jack Sparrow.
- **Pink Panther**: Die Musik des Pink Panther.
- **Die Simpsons**: Musik aus den Simpsons.

**Verwenden** :

Wenn Sie Musik abspielen möchten, ohne sie zu komponieren.
