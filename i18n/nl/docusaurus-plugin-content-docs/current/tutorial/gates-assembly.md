# Bouw obstakels

Dat is alles, uw robot is gemonteerd? Wat als ik je vertelde dat er misschien nog een verrassing op de bodem van je doos zit!

Til de bovenste kartonnen vellen op om een ​​laatste vel met afgeronde hoeken aan de onderkant van het karton te vinden.

![alt-tekst](/img/tutorial/gates-assembly/71490697-3D6C-4291-BBF0-2768DA5ADE4C_1_102_a.jpeg)

Neem dit bord en scheid het in tweeën.

![alt-tekst](/img/tutorial/gates-assembly/9046FC62-53AD-45C0-8F4D-DE7E04C9D8B7_1_201_a-2048x1536.jpeg)

Maak vervolgens de twee elementen in het midden van elke halve plaat los, dit zijn de voeten.
Het andere “U”-vormige element is de deur

![alt-tekst](/img/tutorial/gates-assembly/B7E39673-4AF7-4B1D-9C91-0DD8B087FE19_1_102_a.jpeg)

Monteer de poten op de deur

![alt-tekst](/img/tutorial/gates-assembly/C93D9ED2-DB41-4868-9806-7753AC228BA6_1_102_a.jpeg)

En voila! Je hebt zojuist twee kleine deuren gemaakt waar Eliobot doorheen kan bewegen.

![alt-tekst](/img/tutorial/gates-assembly/47A67EA2-FC0A-4CC5-B111-84B9F697D888_1_102_a.jpeg)

Je kunt hiervan ook profiteren om je obstakels te personaliseren

![alt-tekst](/img/tutorial/gates-assembly/CC591BFB-BC38-47A7-9708-5E2B2DA3F492_1_201_a-2048x1536.jpeg)

Ontdek in de volgende tutorials hoe u Eliobot kunt programmeren om door deze deuren te gaan.
