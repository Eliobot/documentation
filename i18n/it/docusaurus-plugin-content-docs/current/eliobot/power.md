## Utilizzare con Eliobloc

### Attiva Eliobot

Per accendere Eliobot è sufficiente posizionare il pulsante nella posizione `ON`.

### Disattiva Eliobot

Per spegnerlo, stesso principio, è sufficiente posizionare il pulsante nella posizione `OFF`.

## Da utilizzare con Python

Quando Eliobot è collegato ad un computer, è acceso a metà, puoi fargli eseguire comandi, caricargli programmi, li eseguirà, ma tutti gli elementi collegati alla batteria (motori) non verranno alimentati.

Per beneficiare di tutte le funzionalità di Eliobot è necessario posizionare il pulsante di accensione in modalità `ON`.
