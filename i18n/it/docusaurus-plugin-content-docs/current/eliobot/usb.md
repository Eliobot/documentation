La presa USB tipo C permette sia di programmare che di ricaricare Eliobot utilizzando il cavo fornito nella confezione.

Quando è collegato, noterai un piccolo LED che si accende per farti sapere che è in carica, poi si spegne una volta completata la ricarica.
