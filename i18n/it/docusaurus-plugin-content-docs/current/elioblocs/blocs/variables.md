Le variabili consentono di memorizzare valori per riutilizzarli successivamente nel programma. Sono molto utili per archiviare dati di tutti i tipi.

## Crea una variabile

![Creazione variabile](/img/elioblocs/blocs/variables/creer-variable.png)

Per creare una variabile è sufficiente fare clic sul pulsante e si aprirà una finestra in cui inserire il nome della variabile.

![Impostazione nome variabile](/img/elioblocs/blocs/variables/nom-variable.png)

Dategli il nome che preferite, deve essere unico.

## Chiama una variabile

![Utilizzo di una variabile](/img/elioblocs/blocs/variables/appel-variable.png)

**Tipo**: valore

**Definizione**:

Una variabile può essere chiamata ovunque nel programma, purché sia ​​stata dichiarata in precedenza.

**Utilizzo** :

Se vogliamo utilizzare una variabile per memorizzare un valore e riutilizzarlo in seguito.

## Modifica una variabile

![Impostazioni variabili](/img/elioblocs/blocs/variables/change-variable.png)

**Definizione**:

Possiamo:
- Modificare il nome della variabile
- Elimina variabile

**Utilizzo** :

Se vogliamo modificare una variabile già creata.

### Impostato su un valore specifico

![Imposta variabile su valore specifico](/img/elioblocs/blocs/variables/definir-variable.png)

**Tipo**: Ordine

**Definizione**:

Imposta una variabile su un valore specifico. Questo valore può essere un numero, un testo, un elenco, ecc.

**Utilizzo** :

Se vuoi definire una variabile con un valore, ad esempio per ricordare la posizione di un elemento in un elenco.

### Incremento

![Incremento variabile](/img/elioblocs/blocs/variables/incrementer-variable.png)

**Tipo**: Ordine

**Definizione**:

L'incremento di una variabile viene utilizzato per aumentare il valore di una variabile del valore desiderato.

**Utilizzo** :

Se vogliamo aumentare una variabile di 1 ad ogni giro del ciclo, ad esempio.
