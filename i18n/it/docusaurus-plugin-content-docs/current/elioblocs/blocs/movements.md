I blocchi di movimento consentono a Eliobot di muoversi nello spazio.

# Blocchi di movimento per quadrato

## Vai avanti di X spazi

![Sposta X custodie](/img/elioblocs/blocs/movements/avancer-cases.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco permette ad Eliobot di avanzare di un certo numero di caselle.

**Utilizzo** :

Se vuoi far avanzare Eliobot di una certa distanza.

## Gira a destra o a sinistra

![Gira a destra o a sinistra](/img/elioblocs/blocs/movements/tourner-cases.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco permette ad Eliobot di girare a destra o a sinistra (90 gradi).

**Utilizzo** :

Se vuoi eseguire Eliobot.

# Blocchi di movimento libero

## Vai avanti

![Sposta avanti o indietro](/img/elioblocs/blocs/movements/deplacer.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco ti permette di muovere Eliobot avanti o indietro.

**Utilizzo** :

Se vuoi muovere Eliobot avanti o indietro.

## Ruota a destra

![Gira liberamente](/img/elioblocs/blocs/movements/pivoter.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco permette ad Eliobot di ruotare a destra o a sinistra (90 gradi).

**Utilizzo** :

Se vuoi eseguire Eliobot.


## Ruota di un angolo predefinito

![Ruota in un angolo](/img/elioblocs/blocs/movements/pivoter-degres.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco permette a Eliobot di ruotare secondo un angolo preciso.

**Utilizzo** :

Se vuoi fai girare leggermente Eliobot a destra o sinistra.

## Spegnete i motori

![Arrestare i motori](/img/elioblocs/blocs/movements/arreter-moteurs.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco permette di fermare i motori di Eliobot.

**Utilizzo** :

Se vuoi fermare Eliobot.

## Gira una singola ruota

![Gira solo una ruota](/img/elioblocs/blocs/movements/tourner-roue.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco ti permette di girare una singola ruota di Eliobot.

**Utilizzo** :

Se vuoi far girare solo una ruota di Eliobot. È un'alternativa alla rotazione.

## Cambia velocità

![Cambia velocità](/img/elioblocs/blocs/movements/modifier-vitesse.png)

**Tipo**: Ordine

**Definizione**:

Questo blocco ti permette di modificare la velocità di Eliobot (in percentuale).

:::note
100% è la velocità massima.
:::

**Utilizzo** :

Se vuoi che Eliobot vada più veloce o più lento. Ad esempio, rallenta se un sensore rileva un ostacolo.
