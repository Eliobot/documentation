Gli elenchi consentono di memorizzare più valori in un'unica variabile. Sono molto utili per memorizzare dati dello stesso tipo.

## Crea un elenco

### Elenco vuoto

![Crea elenco](/img/elioblocs/blocs/lists/creer-liste-vide.png)

**Tipo**: valore

**Definizione**:

Crea un elenco vuoto.
L'ingranaggio ti consente di aggiungere elementi all'elenco.

**Utilizzo** :

Se vogliamo creare un elenco per memorizzare i valori.

### Elenco con elementi definiti

![Crea elenco con elementi](/img/elioblocs/blocs/lists/creer-liste.png)

**Tipo**: valore

**Definizione**:

Crea un elenco con gli elementi definiti.
L'ingranaggio ti consente di aggiungere più elementi all'elenco.

**Utilizzo** :

Se vogliamo creare una lista con valori già definiti.

### Elenco con elemento ripetuto

![Crea elenco ripetuto](/img/elioblocs/blocs/lists/creer-liste-repete.png)

**Tipo**: valore

**Definizione**:

Crea un elenco con un elemento ripetuto un certo numero di volte.

**Utilizzo** :

Se vogliamo creare una lista con un elemento ripetuto più volte.

### Elenco dal testo

![Crea elenco da testo](/img/elioblocs/blocs/lists/liste-texte.png)

**Tipo**: valore

**Definizione**:

Consente di creare un elenco dal testo. Il separatore permette di definire come vengono separati gli elementi della lista.

**Utilizzo** :

Se desideri creare un elenco da un testo o un CSV.

> #### Esempio
> 
> Il testo: "Sono un programmatore"
> 
> Con il separatore: " " (spazio)
> 
> Fornisce l'elenco: ['Io', 'sono', 'a', 'programmatore']


## Scopri la lunghezza di un elenco

![Lunghezza elenco](/img/elioblocs/blocs/lists/long-liste.png)

**Tipo**: valore

**Definizione**:

Fornisce il numero di elementi presenti in una lista.

**Utilizzo** :

Se vogliamo eseguire un'azione un certo numero di volte a seconda della lunghezza della lista.

## Verifica se una lista è vuota

![Lista vuota](/img/elioblocs/blocs/lists/liste-vide.png)

**Tipo**: blocco booleano

**Definizione**:

Ti permette di verificare se una lista è vuota o meno (restituisce un valore booleano).

**Utilizzo** :

Se vogliamo eseguire un'azione se una lista è vuota. Ad esempio, compilandolo.

## Trova un'occorrenza in un elenco

![Trova occorrenza in list](/img/elioblocs/blocs/lists/occurence-liste.png)

**Tipo**: valore

**Definizione**:

Restituisce la posizione della prima occorrenza di un elemento in un elenco.

**Utilizzo** :

Se vogliamo sapere se un elemento è presente in una lista e in quale posizione.

## Modifica un elenco

![Cambia elemento elenco](/img/elioblocs/blocs/lists/mettre-liste.png)

**Tipo**: Ordine

**Definizione**:

Permette di spostare/inserire un elemento in una determinata posizione nell'elenco.

**Utilizzo** :

Se desideri modificare un elenco spostando un elemento o inserendo un elemento in una determinata posizione.

## Ordina un elenco

![Ordina elenco](/img/elioblocs/blocs/lists/trier-liste.png)

**Tipo**: valore

**Definizione**:

Consente di ordinare un elenco in ordine crescente o decrescente.
Con le opzioni:
- **numeric**: ordina gli elementi come numeri.
- **alfabetico**: ordina gli elementi come testo.
- **ignorando maiuscole e minuscole in ordine alfabetico**: ordina gli elementi come testo senza tenere conto delle lettere maiuscole e minuscole.

**Utilizzo** :

Se vuoi ordinare un elenco per sfogliarlo più facilmente.
