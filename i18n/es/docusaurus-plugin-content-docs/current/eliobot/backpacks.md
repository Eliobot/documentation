Las mochilas Eliobot son módulos que permiten añadir funcionalidades a Eliobot.
Existen varios modelos, cada uno con una función específica.

## Uso con Elioblocs

Todos los bloques asociados con bolsas se agrupan en la categoría <img src={require("@site/static/img/eliobot/backpacks/category-backpacks.jpg").default} style={{ width: "14%", verticalAlign: "middle" }} alt="categoría backpacks" />.

## Usar con Python

Cada mochila utiliza uno o más pines para funcionar y puede utilizar diferentes protocolos de comunicación.
Puedes encontrar la documentación de tu mochila aquí: [Documentación de mochila](/docs/eliobot/backpacks).
