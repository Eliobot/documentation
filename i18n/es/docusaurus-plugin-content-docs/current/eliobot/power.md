## Uso con Elioblocs

### Activa Eliobot

Para encender Eliobot, simplemente coloque el botón en la posición `ON`.

### Apagar Eliobot

Para apagarlo, el mismo principio, simplemente coloque el botón en la posición `OFF`.

## Usar con Python

Cuando Eliobot está conectado a una computadora, está medio encendida, puedes hacer que ejecute comandos, cargarle programas, los ejecutará, pero todos los elementos conectados a la batería (motores) no estarán alimentados.

Para beneficiarte de todas las funciones de Eliobot, debes poner el botón de encendido en modo `ON`.
