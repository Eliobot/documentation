Así está compuesta la interfaz. Esta parte de la explicación te será útil para el resto del tutorial.

![Interfaz Elioblocks](/img/elioblocs/interface/interface.jpg)

> 1. La caja de herramientas (el espacio donde encontrarás todos los bloques necesarios para la programación).
> 
> 2. El espacio de trabajo (aquí es donde puedes armar tus bloques).
> 
> 3. Pestañas que te permiten acceder a tutoriales, ayuda, así como ver el código que se escribirá cuando coloques bloques.
>     
> 4. Los botones:
> - Borrar todo: para limpiar tu espacio de trabajo con un solo clic.
> - Importar: abre un programa previamente guardado.
> - Guardar: guarda tu programa en tu computadora.
> - Conectar: ​​para conectar Eliobot a su computadora.
> - Subir: para enviar el código a Eliobot.
> 
> 5. La terminal (aquí es donde puedes ver mensajes de error o éxito de tu programa y comunicarte con Eliobot).
