Los bloques de movimiento permiten a Eliobot moverse en el espacio.

# Bloques de movimiento por cuadrado

## Avanzar X espacios

![Mover X casos](/img/elioblocs/blocs/movements/avancer-cases.png)

**Tipo**: Orden

**Definición** :

Este bloque permite a Eliobot avanzar una determinada cantidad de casillas.

**Usar** :

Si quieres hacer avanzar a Eliobot una cierta distancia.

## Girar a la derecha o a la izquierda

![Gire a la derecha o a la izquierda](/img/elioblocs/blocs/movements/tourner-cases.png)

**Tipo**: Orden

**Definición** :

Este bloque le permite a Eliobot girar a la derecha o a la izquierda (90 grados).

**Usar** :

Si desea ejecutar Eliobot.

# Bloques de libre movimiento

## Avanzar

![Avanzar o retroceder](/img/elioblocs/blocs/movements/deplacer.png)

**Tipo**: Orden

**Definición** :

Este bloque te permite mover Eliobot hacia adelante o hacia atrás.

**Usar** :

Si quieres mover Eliobot hacia adelante o hacia atrás.

## Girar hacia la derecha

![Gira libremente](/img/elioblocs/blocs/movements/pivoter.png)

**Tipo**: Orden

**Definición** :

Este bloque permite a Eliobot girar hacia la derecha o hacia la izquierda (90 grados).

**Usar** :

Si desea ejecutar Eliobot.


## Girar en un ángulo predefinido

![Gira hacia un ángulo](/img/elioblocs/blocs/movements/pivoter-degres.png)

**Tipo**: Orden

**Definición** :

Este bloque permite a Eliobot girar en un ángulo preciso.

**Usar** :

Si quieres hacer que Eliobot gire ligeramente hacia la derecha o hacia la izquierda.

## Detener los motores

![Detén los motores](/img/elioblocs/blocs/movements/arreter-moteurs.png)

**Tipo**: Orden

**Definición** :

Este bloque te permite detener los motores de Eliobot.

**Usar** :

Si quieres detener a Eliobot.

## Gira una sola rueda

![Gira solo una rueda](/img/elioblocs/blocs/movements/tourner-roue.png)

**Tipo**: Orden

**Definición** :

Este bloque te permite girar una sola rueda de Eliobot.

**Usar** :

Si quieres hacer girar solo una rueda de Eliobot. Es una alternativa a la rotación.

## Cambiar velocidad

![Cambiar la velocidad](/img/elioblocs/blocs/movements/modifier-vitesse.png)

**Tipo**: Orden

**Definición** :

Este bloque te permite modificar la velocidad de Eliobot (en porcentaje).

:::note
100% es la velocidad máxima.
:::

**Usar** :

Si quieres que Eliobot vaya más rápido o más lento. Por ejemplo, reducir la velocidad si un sensor detecta un obstáculo.
