Las variables le permiten almacenar valores para reutilizarlos más adelante en el programa. Son muy útiles para almacenar datos de todo tipo.

## Crear una variable

![Creación de variables](/img/elioblocs/blocs/variables/creer-variable.png)

Para crear una variable, simplemente haga clic en el botón y se abrirá una ventana para ingresar el nombre de la variable.

![Configuración de nombre de variable](/img/elioblocs/blocs/variables/nom-variable.png)

Ponle el nombre de tu elección, debe ser único.

## Llamar a una variable

![Usando una variable](/img/elioblocs/blocs/variables/appel-variable.png)

**Tipo**: Valor

**Definición** :

Se puede llamar a una variable en cualquier parte del programa, siempre que haya sido declarada anteriormente.

**Usar** :

Si queremos usar una variable para almacenar un valor y reutilizarlo más tarde.

## Modificar una variable

![Configuración variable](/img/elioblocs/blocs/variables/change-variable.png)

**Definición** :

Podemos:
- Cambiar el nombre de la variable.
- Eliminar variable

**Usar** :

Si queremos modificar una variable ya creada.

### Establecer un valor específico

![Establecer variable en un valor específico](/img/elioblocs/blocs/variables/definir-variable.png)

**Tipo**: Orden

**Definición** :

Establezca una variable en un valor específico. Este valor puede ser un número, texto, lista, etc.

**Usar** :

Si desea definir una variable a un valor, por ejemplo para recordar la posición de un elemento en una lista.

### Incremento

![Incremento variable](/img/elioblocs/blocs/variables/incrementer-variable.png)

**Tipo**: Orden

**Definición** :

Incrementar una variable se utiliza para aumentar el valor de una variable en el valor deseado.

**Usar** :

Si queremos aumentar una variable en 1 en cada vuelta del bucle por ejemplo.
