Rucsacii Eliobot sunt module care vă permit să adăugați funcționalități la Eliobot.
Există mai multe, fiecare cu o funcționalitate specifică.

## Utilizați cu Elioblocs

Toate blocurile asociate cu rucsacuri sunt grupate în categoria <img src={require("@site/static/img/eliobot/backpacks/category-backpacks.jpg").default} style={{ width: "14%", verticalAlign: "middle" }} alt="categorie rucsacuri" />.

## Utilizați cu Python

Fiecare rucsac folosește unul sau mai mulți pini pentru a funcționa și poate folosi diferite protocoale de comunicare.
Puteți găsi documentația pentru rucsac aici: [Documentație rucsac](/docs/eliobot/backpacks).
