Priza USB tip C vă permite atât să programați, cât și să reîncărcați Eliobot folosind cablul furnizat în cutie.

Când este conectat, veți observa un mic LED care se aprinde pentru a vă anunța că se încarcă, apoi se stinge odată ce încărcarea este completă.
