Acesta este modul în care este compusă interfața. Această parte a explicației vă va fi utilă pentru restul tutorialului.

![Interfața Elioblocks](/img/elioblocs/interface/interface.jpg)

> 1. Cutia de instrumente (spațiul în care veți găsi toate blocurile necesare pentru programare).
> 
> 2. Spațiul de lucru (aici vă puteți asambla blocurile).
> 
> 3. File care vă permit să accesați tutoriale, ajutor, precum și să vedeți codul care va fi scris când plasați blocuri.
>     
> 4. Butoanele:
> - Ștergeți totul: pentru a vă curăța spațiul de lucru cu un singur clic.
> - Import: deschide un program salvat anterior.
> - Salvare: salvați programul pe computer.
> - Conectare: pentru a conecta Eliobot la computer.
> - Încărcare: pentru a trimite codul către Eliobot.
> 
> 5. Terminalul (aici puteți vedea mesajele de eroare sau de succes din programul dvs. și puteți comunica cu Eliobot).
