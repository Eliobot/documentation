# Introducere

Să descoperim **Docusaurus în mai puțin de 5 minute**.

## Noțiuni de bază

Începeți prin **crearea unui site nou**.

Sau **încercați imediat Docusaurus** cu **[docusaurus.new](https://docusaurus.new)**.

### De ce vei avea nevoie

- [Node.js](https://nodejs.org/en/download/) versiunea 18.0 sau mai recentă:
- Când instalați Node.js, vi se recomandă să bifați toate casetele de selectare legate de dependențe.

## Generați un site nou

Generați un nou site Docusaurus folosind **șablonul clasic**.

Șablonul clasic va fi adăugat automat în proiect după ce executați comanda:

```bash
npm init docusaurus@latest my-website classic
```

Puteți introduce această comandă în Command Prompt, Powershell, Terminal sau orice alt terminal integrat al editorului de cod.

Comanda instalează, de asemenea, toate dependențele necesare de care aveți nevoie pentru a rula Docusaurus.

## Începeți-vă site-ul

Rulați serverul de dezvoltare:

```bash
cd my-website
npm run start
```

Comanda `cd` schimbă directorul cu care lucrați. Pentru a lucra cu site-ul Docusaurus nou creat, va trebui să navigați în terminalul de acolo.

Comanda `npm run start` creează site-ul dvs. web local și îl servește printr-un server de dezvoltare, gata pentru a fi vizualizat la http://localhost:3000/.

Deschideți `docs/intro.md` (această pagină) și editați câteva rânduri: site-ul **se reîncarcă automat** și vă afișează modificările.
