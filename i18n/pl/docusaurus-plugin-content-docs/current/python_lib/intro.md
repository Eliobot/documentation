# Wstęp

Odkryjmy **Dokuzaura w mniej niż 5 minut**.

## Pierwsze kroki

Zacznij od **utworzenia nowej witryny**.

Lub **wypróbuj natychmiast Docusaurus** za pomocą **[docusaurus.new](https://docusaurus.new)**.

### Czego będziesz potrzebować

- [Node.js](https://nodejs.org/en/download/) wersja 18.0 lub nowsza:
- Podczas instalacji Node.js zaleca się zaznaczenie wszystkich pól wyboru związanych z zależnościami.

## Wygeneruj nową witrynę

Wygeneruj nową witrynę Docusaurus przy użyciu **klasycznego szablonu**.

Klasyczny szablon zostanie automatycznie dodany do Twojego projektu po uruchomieniu komendy:

```bash
npm init docusaurus@latest my-website classic
```

Możesz wpisać to polecenie w wierszu poleceń, Powershell, terminalu lub dowolnym innym zintegrowanym terminalu edytora kodu.

Polecenie instaluje również wszystkie niezbędne zależności potrzebne do uruchomienia Docusaurus.

## Uruchom swoją witrynę

Uruchom serwer deweloperski:

```bash
cd my-website
npm run start
```

Komenda `cd` zmienia katalog, z którym pracujesz. Aby pracować z nowo utworzoną witryną Docusaurus, musisz nawigować po znajdującym się tam terminalu.

Polecenie `npm run start` tworzy lokalnie witrynę internetową i udostępnia ją za pośrednictwem serwera programistycznego, gotową do przeglądania pod adresem http://localhost:3000/.

Otwórz `docs/intro.md` (tę stronę) i edytuj kilka linijek: strona **przeładowuje się automatycznie** i wyświetla Twoje zmiany.
