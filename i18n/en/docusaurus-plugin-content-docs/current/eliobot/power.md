## Use with Elioblocs

### Turn on Eliobot

To turn on Eliobot, simply position the button in the `ON` position.

### Turn off Eliobot

To turn it off, same principle, simply position the button in the `OFF` position.

## Use with Python

When Eliobot is connected to a computer, it is half on, you can make it execute commands, upload programs to it, it will execute them, but all the elements connected to the battery (motors) will not be powered.

To benefit from all the features of Eliobot, you must put the power button in `ON` mode.
