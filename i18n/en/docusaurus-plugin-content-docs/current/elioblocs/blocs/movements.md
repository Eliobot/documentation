Movement blocks let you move Eliobot around.

# Movement blocks per square

## Move forward X spaces

![Move X cases](/img/elioblocs/blocs/movements/avancer-cases.png)

**Type**: Order

**Definition** :

This block allows Eliobot to advance a certain number of squares.

**Use** :

If you want to move Eliobot forward a certain distance.

## Turn right or left

![Turn right or left](/img/elioblocs/blocs/movements/tourner-cases.png)

**Type**: Order

**Definition** :

This block allows Eliobot to turn right or left (90 degrees).

**Use** :

If you want to run Eliobot.

# Free movement blocks

## Move forward

![Move forward or backward](/img/elioblocs/blocs/movements/deplacer.png)

**Type**: Order

**Definition** :

This block allows you to move Eliobot forward or backward.

**Use** :

If you want to move Eliobot forward or backward.

## Swivel right

![Turn freely](/img/elioblocs/blocs/movements/pivoter.png)

**Type**: Order

**Definition** :

This block allows Eliobot to rotate to the right or left (90 degrees).

**Use** :

If you want to run Eliobot.


## Rotate by a predefined angle

![Turn to an angle](/img/elioblocs/blocs/movements/pivoter-degres.png)

**Type**: Order

**Definition** :

This block allows Eliobot to rotate at a precise angle.

**Use** :

If you want to make Eliobot turn slightly to the right or left.

## Stop the engines

![Stop the motors](/img/elioblocs/blocs/movements/arreter-moteurs.png)

**Type**: Order

**Definition** :

This block allows you to stop Eliobot's engines.

**Use** :

If you want to stop Eliobot.

## Spin a single wheel

![Spin ​​only one wheel](/img/elioblocs/blocs/movements/tourner-roue.png)

**Type**: Order

**Definition** :

This block allows you to turn a single Eliobot wheel.

**Use** :

If you want to spin just one Eliobot wheel. It is an alternative to rotation.

## Change speed

![Change the speed](/img/elioblocs/blocs/movements/modifier-vitesse.png)

**Type**: Order

**Definition** :

This block allows you to modify Eliobot's speed (in percentage).

:::note
100% is the maximum speed.
:::

**Use** :

If you want Eliobot to go faster or slower. For example, slow down if a sensor detects an obstacle.
