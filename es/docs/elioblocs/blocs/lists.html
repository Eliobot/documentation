Listas

Las listas le permiten almacenar múltiples valores en una sola variable. Son muy útiles para almacenar datos del mismo tipo.

Crear una lista​

Lista vacía​

Tipo: Valor

Definición :

Crea una lista vacía. El engranaje le permite agregar elementos a la lista.

Usar :

Si queremos crear una lista para almacenar valores.

Lista con elementos definidos​

Tipo: Valor

Definición :

Crea una lista con los elementos definidos. El engranaje le permite agregar más elementos a la lista.

Usar :

Si queremos crear una lista con valores ya definidos.

Lista con elemento repetido​

Tipo: Valor

Definición :

Crea una lista con un elemento repetido un número determinado de veces.

Usar :

Si queremos crear una lista con un elemento repetido varias veces.

Lista de texto​

Tipo: Valor

Definición :

Le permite crear una lista a partir de texto. El separador permite definir cómo se separan los elementos de la lista.

Usar :

Si desea crear una lista a partir de un texto o un CSV.

Ejemplo​

El texto: "Soy programador"

Con el separador: " " (espacio)

Da la lista: ['Yo', 'soy', 'un', 'programador']

Descubra la longitud de una lista​

Tipo: Valor

Definición :

Da el número de elementos presentes en una lista.

Usar :

Si queremos hacer una acción un número determinado de veces dependiendo de la longitud de la lista.

Probar si una lista está vacía​

Tipo: bloque booleano

Definición :

Le permite probar si una lista está vacía o no (devuelve un valor booleano).

Usar :

Si queremos realizar una acción si una lista está vacía. Por ejemplo, completándolo.

Buscar una ocurrencia en una lista​

Tipo: Valor

Definición :

Devuelve la posición de la primera aparición de un elemento en una lista.

Usar :

Si queremos saber si un elemento está presente en una lista y en qué posición.

Editar una lista​

Tipo: Orden

Definición :

Le permite mover/insertar un elemento a una posición determinada en la lista.

Usar :

Si desea modificar una lista moviendo un elemento o insertando un elemento en una posición determinada.

Ordenar una lista​

Tipo: Valor

Definición :

Le permite ordenar una lista en orden ascendente o descendente. Con las opciones:

  • numérico: ordena elementos como números.
  • alfabético: ordena elementos como texto.
  • ignorando alfabéticamente mayúsculas y minúsculas: ordena los elementos como texto sin tener en cuenta las letras mayúsculas y minúsculas.

Usar :

Si desea ordenar una lista para navegarla más fácilmente.
