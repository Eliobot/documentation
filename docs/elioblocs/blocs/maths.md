- soustraction
- multiplication
- division
- puissance

![Constraint a number](../../../static/img/elioblocs/blocs/maths/contraindre-chiffre.png)

## atan2

![atan2](../../../static/img/elioblocs/blocs/maths/atan2.png)

Ce bloc permet de calculer l'arc tangente de deux nombres.
